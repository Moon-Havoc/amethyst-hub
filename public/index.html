Access rules

What the backend checks

Normal

24-hour renewal cycle

Normal keys expire after one day and can be regenerated from the portal.

Premium

Lifetime access

Premium keys issued with `!prem-gen` never expire and are delivered through Discord.

Verify

Username-bound validation

The Roblox GUI only passes if the submitted key belongs to that exact Roblox username.

Launch notes

Before you ship

Discord first

Staff should use Discord for premium generation, revokes, bans, kicks, and moderation actions.

Website second

This page is the self-serve normal key portal for users who only need a standard 24-hour key.

Roblox integration

The in-game GUI points at the live Railway deployment and copies the site link with the player name prefilled for a smoother request flow.

Discord ops

Command surface

The bot handles key access, blacklists, moderation, and role management in one command set.

Key issuing

!gen-key {user} {robloxuser} !prem-gen {user} {robloxuser} !revoke_key {user} {key} !revoke_prem {user} {key} !reset_hwid

Trust & blacklist

!blacklist {user} {reason} !blacklists !unblacklist {user} !commands !check-perms

Moderation

!ban {user} {reason} !kick {user} {reason} !mute {user} {duration} {reason} !unban {user} !unmute {user} !role {user} {role}
